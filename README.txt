Install CalcSignature.exe as "sendto" mechanism of Windows Explorer:

1. Copy CalcSignature.exe to desired location
2. Open Windows Explorer
   2.1 Type "shell:sendo" in address bar
   2.2 In opened main window right-click and create a new shortcut, which points to placed CalcSignature.exe

Now it is possible to right-click any file, then press sendto CalcSignature, in order to open it with the selected file
